📋 Instruction File Format:
Each row defines a rule for one column.
Column A: Column name (e.g. Weight)
Column B: Rule (e.g. must be numeric with unit like 200g or 500ml)

Example rules:
Weight | must contain a number followed by g, kg, ml or L
Brand | must not be empty
Flavour | must be one of: Original, Mint, Berry, Lemon
Country | must be a valid EU or UK country name
Barcode | must be 13 digits
Category | must not be empty
